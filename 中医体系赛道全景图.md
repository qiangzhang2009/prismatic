## 一、临床中医学

面向患者的诊疗实践学科群

| 三级赛道 | 四级细分 |
|---------|---------|
| **中医内科** | 心系病证（胸痹、心悸、不寐）、肝系病证（郁证、胁痛、眩晕）、脾胃病证（胃痛、泄泻、便秘）、肺系病证（咳嗽、哮病、喘证）、肾系病证（水肿、淋证、癃闭）、气血津液病证（汗证、消渴、瘿病）、肢体经络病证（痹证、痿证、腰痛） |
| **中医外科** | 疮疡（疖、痈、疔、丹毒）、乳房病（乳痈、乳癖、乳岩）、瘿瘤岩（瘿病、瘤、岩证）、肛肠病（痔、肛痈、肛裂、脱肛）、泌尿男性病（淋证、浊证、阳痿）、外伤与周围血管病（疮疡、压疮、糖尿病足） |
| **中医妇科** | 月经病（经行诸证、经间期出血、闭经、崩漏）、带下病（带下过多、过少）、妊娠病（恶阻、胎动不安、滑胎）、产后病（产后血晕、产后腹痛、产后发热、缺乳）、妇科杂病（癥瘕、不孕症、阴挺） |
| **中医儿科** | 肺系病证（感冒、咳嗽、肺炎喘嗽）、脾系病证（泄泻、厌食、积滞）、心肝系病证（惊风、癫痫、多动症）、肾系病证（水肿、遗尿、尿频）、传染病（麻疹、水痘、痄腮、手足口病） |
| **中医皮肤科** | 病毒性皮肤病（扁平疣、寻常疣、单纯疱疹）、细菌性皮肤病（脓疱疮、毛囊炎）、真菌性皮肤病（手足癣、甲癣）、动物性皮肤病（疥疮、虫咬皮炎）、皮炎湿疹类（湿疹、接触性皮炎、药疹）、瘙痒性皮肤病（瘙痒症、神经性皮炎）、色素性皮肤病（白癜风、黄褐斑）、皮肤附属器病（痤疮、脱发） |
| **中医骨伤科** | 骨折（各部位骨折整复与固定）、脱位（肩/肘/髋等关节脱位）、筋伤（扭伤、挫伤、劳损）、创伤（皮肉伤、出血、创伤急救）、骨病（骨质疏松、骨关节炎、先天畸形） |
| **中医眼科** | 胞睑病（睑缘炎、睑腺炎、睑板腺囊肿）、白睛病（结膜炎、巩膜炎）、黑睛病（角膜炎、角膜溃疡）、瞳神病（虹膜睫状体炎、青光眼、白内障）、晶珠病（白内障）、神膏病（玻璃体混浊）、目眶病（眼眶炎症、肿瘤） |
| **中医耳鼻喉科** | 耳科（耳鸣耳聋、中耳炎、眩晕）、鼻科（鼻炎、鼻窦炎、鼻出血）、咽喉科（喉痹、乳蛾、喉喑、梅核气） |
| **中医急诊科** | 急性心脑血管病中医参与、急性中毒救治、发热待查、外感高热、厥脱证救治 |

## 二、中药学

中药从资源到应用的完整学科链

| 三级赛道 | 四级细分 |
|---------|---------|
| **中药资源学** | 道地药材研究（川药、广药、云药、贵药、浙药等）、野生药材资源调查、栽培技术与规范化种植、珍稀濒危药材保护、中药材产地加工 |
| **中药鉴定学** | 形态鉴定（基原鉴定、性状鉴定）、显微鉴定（组织切片、微量化学）、理化鉴定（化学反应、薄层色谱）、DNA条形码鉴定、指纹图谱技术 |
| **中药炮制学** | 净制（挑选、筛选、风选、水选）、切制（片、段、块、丝）、炒制（清炒、麸炒、米炒）、炙法（蜜炙、酒炙、醋炙、盐炙、姜炙）、煅法（明煅、煅淬、闷煅）、蒸煮燀制、其他制法（发芽、发酵、制霜） |
| **中药化学** | 生物碱类成分、黄酮类成分、萜类和挥发油、皂苷类成分、糖类与多糖、醌类成分、苯丙素类、鞣质类、其他活性成分 |
| **中药药理学** | 中药药效学（抗炎、免疫、抗肿瘤等）、中药药动学（吸收、分布、代谢）、中药毒理学（急性毒性、慢性毒性）、分子药理学（信号通路、基因调控）、血清药理学、网络药理学 |
| **中药制剂学** | 固体制剂（丸剂、片剂、颗粒剂、胶囊剂）、液体制剂（合剂、口服液、注射剂）、半固体制剂（膏剂、凝胶剂）、中药新剂型（缓释制剂、靶向制剂、经皮制剂、速释制剂）、制剂辅料研究 |
| **中药质量控制** | 指纹图谱与特征图谱、多成分含量测定、重金属检测、农药残留检测、黄曲霉毒素检测、微生物限度检查、杂质检查 |
| **临床中药学** | 中药药性理论、药物配伍禁忌（十八反、十九畏）、中药不良反应监测、临床药学服务、中药合理用药、药食同源研究 |
| **中药药代动力学** | 吸收机制研究、代谢产物分析、药物相互作用、PK/PD模型 |

## 三、针灸推拿学

非药物外治法技术体系

| 三级赛道 | 四级细分 |
|---------|---------|
| **经络腧穴学** | 十二经脉循行与病候、奇经八脉分布与功能、十五别络与经别、经筋分布与病候、皮部理论、腧穴定位方法（骨度分寸、体表标志、手指比量）、腧穴主治特性、特定穴（井荥输经合、俞募穴、原络穴、郄穴等） |
| **针法灸法学** | 毫针基本手法（进针、留针、出针）、毫针补泻手法（捻转补泻、提插补泻）、三棱针法（点刺、散刺、挑刺）、皮肤针法（叩刺）、皮内针法（颗粒式、揿钉式）、电针疗法、灸法（艾炷灸、艾条灸、温针灸、温和灸、雀啄灸、回旋灸）、隔物灸（隔姜、隔蒜、隔盐、隔附子饼）、温灸器灸、天灸 |
| **针灸治疗学** | 内科病证（头痛、面瘫、腰痛、坐骨神经痛）、妇儿科病证（痛经、月经不调、小儿遗尿）、皮外科病证（带状疱疹、湿疹、荨麻疹）、五官科病证（近视、鼻渊、牙痛）、急症（晕厥、急性疼痛、高热） |
| **推拿学** | 成人推拿（全身各部位推拿套路、软组织损伤推拿、正骨推拿）、小儿推拿（常用手法、复式操作、特定穴推拿、常见病治疗）、正骨推拿（关节整复、脊柱调理）、足部反射区推拿 |
| **刮痧拔罐疗法** | 刮痧理论依据与适应证、刮痧手法与部位、拔罐疗法（留罐、闪罐、走罐、刺络拔罐、药罐）、注意事项与禁忌 |
| **特种针灸技术** | 头针疗法（取穴原则、主要穴线）、耳针疗法（耳穴分布、常见病治疗）、眼针疗法、腹针疗法、腕踝针、穴位注射疗法、穴位埋线疗法、穴位贴敷疗法（三伏贴、三九贴）、小针刀疗法、铍针、银质针、杵针 |
| **针灸美容与养生** | 针灸减肥、针灸祛斑、针灸抗衰老、针灸调理亚健康 |

## 四、中医养生学

预防医学与健康管理

| 三级赛道 | 四级细分 |
|---------|---------|
| **食疗药膳学** | 药食同源理论与品种、食物四气五味理论、辨体施膳原则与方法、节气养生膳（二十四节气食疗）、慢性病食疗方案（糖尿病、高血压、痛风等）、美容养颜膳、体重管理食疗 |
| **情志养生** | 七情致病特点与调节、五音疗疾理论、音乐养生处方、情志相胜法、心理疏导技术、禅修与冥想、中医心理治疗 |
| **运动养生** | 太极拳（陈式、杨式、吴式、武式、孙式）、八段锦、五禽戏、易筋经、六字诀、气功（静功、动功）、导引术、华佗五禽戏、散步与慢跑养生 |
| **体质养生** | 九种体质分类标准（平和、气虚、阳虚、阴虚、痰湿、湿热、血瘀、气郁、特禀）、体质辨识技术、个性化养生方案、体质调理膏方、体质与疾病相关性研究 |
| **中医美容** | 面部美容（针灸美容、穴位按摩、面膜美容）、美体塑形（减肥塑身、针灸调理）、毛发养护（乌发、生发）、中医化妆品（中药护肤成分研发）、皮肤亚健康调理 |
| **中医健康管理** | 慢病中医管理（糖尿病、高血压、肿瘤康复）、亚健康评估与干预、妇女周期养生（经期、孕期、围绝经期）、儿童中医健康管理、老年人中医养生、养生馆运营与管理 |
| **环境养生** | 起居养生（作息安排、居室布置）、四季养生（春生夏长秋收冬藏）、地域养生（南北差异、高原养生）、时间养生（时辰经络养生、十二时辰养生法） |

| **各家学说** | 黄帝内经学术思想、伤寒学派（张仲景）、金元四大家（刘完素寒凉派、李杲补土派、张从正攻邪派、朱震亨滋阴派）、温病学派（叶天士、吴鞠通）、经方派、时方派、汇通派 |
| **中医文献学** | 经典文献整理与研究（《黄帝内经》《难经》《伤寒论》《金匮要略》《神农本草经》）、出土医学文献（马王堆帛书、武威汉简）、中医版本学、中医文献数字化、中医古籍校注、中医文献考证 |
| **中医哲学** | 中医象思维、中医系统论、中医天人相应理论、中医生命观、中医伦理思想 |
